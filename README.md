# OPN 首码项目页面

这是一个 OPN 首码项目的单页网站，已优化 SEO，可直接部署到 GitHub Pages。

## 文件说明

- `index.html` - 主页面文件，包含完整的页面内容和样式
- `DEPLOY.md` - GitHub Pages 部署详细指南
- `SEO_GUIDE.md` - SEO 优化和引流策略指南
- `sitemap.xml` - 网站地图，帮助搜索引擎索引
- `robots.txt` - 爬虫规则文件
- `README.md` - 本文件

## 快速开始

1. 修改 `index.html` 中的内容（如需要）
2. 修改 `sitemap.xml` 和 `index.html` 中的 `yourusername` 为你的 GitHub 用户名
3. 按照 `DEPLOY.md` 中的步骤部署到 GitHub Pages

## 主要特性

- ✅ SEO 优化（meta 标签、Open Graph、Schema.org 等）
- ✅ 响应式设计，支持移动端
- ✅ 美观的 UI 界面
- ✅ 完整的项目介绍和注册指引
- ✅ 社交媒体分享优化
