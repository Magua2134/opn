
# 🚀 OPN首码项目 - 快速部署指南

## 第一步：安装Git（如果还没安装）

1. 下载Git：https://git-scm.com/downloads
2. 安装时一路点击"Next"即可
3. 安装完成后重启PowerShell或电脑

## 第二步：创建GitHub仓库

1. 登录GitHub：https://github.com
2. 点击右上角"+" → "New repository"
3. 仓库名称填写：`OPN`
4. 选择 **Public**，不要勾选任何选项
5. 点击 "Create repository"

## 第三步：保存图片（重要！）

把4张图片保存到 `images` 文件夹：
- `opn-project-intro.jpg` - 第4张图（OPN项目介绍）
- `opn-qr-code.jpg` - 第2张图（二维码）
- `opn-app-screenshot.jpg` - 第1张图（应用操作截图）
- `opn-highlights.jpg` - 第3张图（亮点图）

## 第四步：运行部署脚本

### 方法一：使用批处理脚本（推荐）
双击运行 `deploy.bat` 文件，按提示输入信息即可！

### 方法二：手动执行命令

打开PowerShell，进入目录：
```powershell
cd c:\Users\12248\Desktop\OPN
```

然后按顺序执行：
```powershell
git init
git config user.name "Magua2134"
git config user.email "你的GitHub邮箱"
git add .
git commit -m "Initial commit: OPN首码项目页面"
git remote add origin https://github.com/Magua2134/OPN.git
git branch -M main
git push -u origin main
```

## 第五步：启用GitHub Pages

1. 进入你的GitHub仓库：https://github.com/Magua2134/OPN
2. 点击 **Settings**
3. 左侧找到 **Pages**
4. Source选择：**Deploy from a branch**
5. Branch选择：**main**，保存
6. 等待1-5分钟

## 访问网站

部署成功后，你的网站地址是：
**https://Magua2134.github.io/OPN/**

## ❓ 常见问题

**Q: Git推送失败怎么办？**
A: 检查网络、GitHub账号、仓库是否创建成功

**Q: 网站一直显示404？**
A: 等待5-10分钟，GitHub Pages需要时间部署

**Q: 如何更新网站内容？**
A: 修改文件后重新commit和push即可
